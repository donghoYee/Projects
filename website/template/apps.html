1. ADesign: AI Design-tool

1. What Is It?

It's an artificailly guided designing software that creates images based on the user's inputs. Use can be divided into two big parts: 1. The user can draw rough sketches from the begining and turn it into realistic product or sight 2. The user can input existing picture or drawing and change small parts of it as they wish, and turn the existing image into revised one. The above picture shows the practicality of our app.

2. How does It work?

We use Conditioned Denoising Diffusion Probalistic Model(DDPM) as our base machine-learning structure. With the help of Continous Masking developed by us, the software can change only the wanted parts of the existing image, which enhances the practicality of our app and distinguishes it from other leading-technology designing softwares.
User's inputs including initial image, prompt, mask are transmitted to our GPU-server for inference and outputs are transmitted back to the user.

3. What can be Improved?

By the power of our GPU-server, inference usually takes 3 seconds, which is fast but can be improved, and we can make multiple results in the same time if our servers were upgraded, which would greately satisfy the end-user as every results are different due to our models being probalistic.
As technology grows, we will continue to study cutting-edge techniques and ensure that the user gets the best experience using our app.

2. Turn Text into Realistic Image

1. What Is It?

It's an App that creates realistic-looking images based on user's prompts. one can input any prompts, no matter how impossible it may sound.
For example, the above images was generated by our app with prompt "An astronaut riding a horse in the moon". We can fulfil our curiosity of how a crazy-sounding description can come to life by generating actual images corresponding to it, or we can just play it simple and enter normal prompts like "a beautiful mountain with lake and sunsets" and admire the true-looking outputs of the app.

2. How does It work?

The app gets input prompt from users and sends it to our server. Then, our GPU-server uses machine-learned models to generate images from it. We use stable-diffusion model as our generative model, which is included in a family of denoising-diffusion models.
More information about denoising-diffusion models can be seen in Achivements/"Denoising Diffusion Probalistic Models".
The image is than transferred back to the user to display outputs

3. What can be Improved?

Inference time takes up to 5 seconds for now, which can be multipled by the number of users concurrently using the app. We plan to upgrade our GPU-server to quicken our inference time and give better experience to the end-users
We also plan to create an additional app that inputs User's drawings and the way the user wants it to be, and create a beautiful artwork based on it. It also uses stable-diffusion, so It could be an extention of our app.

3. Integral & Graph : Math Tools

1. What Is It?

It's a graphic calculator designed to graph functions and find the area under the graph. It's useful to use in situations where you have some simple function and have to graph it, but you don't want to use expensive applications that are complicated hard to use.
Our app is really intuitive and simple to use. Just put in the function and press "OK"!. Users can input the range of 'x' to specify the region they want to show.
It also integrates the area under the graph, so it's usefull in engineering purposes.

2. How does It work?

We use python's Numpy library to calculate the integration and matplotlib to graph the outputs. Numpy library has tons of functions available to use, so you can basically input all the functions you can think of, and it will work.
For integration, we use Riemann's integration to approximate the solution
Computation is done on our servers, so no need for good hardware on user's side is required.

3. What can be Improved?

There is currently no machine-learning code required on this app, so computation time is really small. However, when inputting a wide range of 'x', computation time does increase to noticeable levels, and we plan to upgrade our hardware to shorten waiting time.
Also, since the app is really simple, there is not much that the user can do. So we plan to upgrade it's functionality and put in more applications!
